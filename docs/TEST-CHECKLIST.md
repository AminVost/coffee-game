# چک‌لیست تست

## رابط عمومی

- صفحه اصلی در عرض 360، 390، 768 و Desktop
- Dark و Light Mode
- فهرست و جزئیات مسابقات
- ثبت یک و چند شرکت‌کننده
- نتایج زنده و رنکینگ

## ورود

- ورود مدیر Mock
- ورود بازیکن Mock
- OTP با کد 123456
- رد رمز یا کد اشتباه
- خروج و حذف Cookie

## MySQL

- Import بدون خطا در phpMyAdmin
- تغییر `DATA_MODE=mysql`
- نمایش مسابقات Seed
- ورود مدیر Seed
- ثبت‌نام و کنترل موبایل تکراری
- Soft Delete مسابقه

## پنل مدیریت

- ساخت مسابقه از قالب
- ویرایش ظرفیت و قیمت
- ثبت نتیجه
- تایید/رد فیش
- ویرایش محتوای صفحات
- مشاهده در موبایل
